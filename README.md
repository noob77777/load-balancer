# load-balancer <br />

## Balance http and socket.io requests <br />
## Dynamically update servers list on the fly.

### `npm install`
Install all dependencies.

### `node app.js`
Runs the app in the development mode.<br />
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

### `node test.js`
Runs the testing script

### `node updateServers.js`
Updates the servers list to match servers.txt.

### Deployment
Deployed with Google Cloud Compute Engine.<br />
Deployment branch is v1.0
